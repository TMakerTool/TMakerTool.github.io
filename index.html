What is TMaker

TMaker is a tool to help terraria modders make basic weapons. Requires TModLoader, base mod, and a png called "sword".

Requirements

  • Base Mod

    Requires Base Mod To Put Weapon In.

  • A PNG

    Requires a PNG File Called "sword".
